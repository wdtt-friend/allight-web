# allight-web

React 19 + Vite SPA integrating with the allight-server API

## 기술 스택

- React 19
- TypeScript
- Vite
- ESLint

### 설치 및 실행

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 미리보기
npm run preview
```

### 개발 스크립트

- `npm run dev` - 개발 서버 실행 (HMR 포함)
- `npm run build` - 프로덕션 빌드
- `npm run lint` - ESLint 검사
- `npm run preview` - 빌드 결과 미리보기
